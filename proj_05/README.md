## Extract Text from Images with Azure AI Vision

This lab showed how to use Azure AI Vision in Vision Studio to extract text from images.

**Key Learnings:**
-   Understanding OCR
-   Azure AI Services Integration
-   Connecting Vision Studio
-   Text Extraction in Action

I tested with three images: two book covers, and one book spine, where the text was potentially upside down. Vision Studio successfully extracted the text from all three images.
